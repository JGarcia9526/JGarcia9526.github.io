Sistema de Control Financiero

Diseño y Desarrollo de Sistema Web para la empresa AEG Systems S.A. de C.V.

Este proyecto, consistio en la implementación de un sistema web para la empresa AEG Systems S.A. de C.V. El objetivo de este Sistema es automatizar al 100% algunos procesos de las área de Contabilidad, Almacén y Finanzas

Algunos de los Módulos existentes dentro de esta plataforma web son:
•Módulo Configuración Organizacional : Gestión de Departamentos, Gestión de Puestos de Trabajo,
•Módulo Configuración Operacional : Gestión de Permisos, Gestión de Roles de Usuarios y Gestión de Acceso a Usuarios Existentes,
•Módulo Configuración Regional : Gestión de Regiones, Gestión de Paises, Gestión de Estados, Gestión de Tipos de Monedas, Gestión de Sectores Industriales,
•Módulo Planeaciones : Generación de Presupuesto por Proyecto , Carga Masiva de Presupuesto por proyecto, Historico de Presupuestos Generados por proyecto, Consulta de Presupuestos por Proyecto Cerrados,
•Módulo Reportes : Consulta de Reportes de Proyectos, Exportación de Reportes de Proyectos, Actualizacion de Reportes de Proyectos Valor Ganado,
•Módulo Viaticos : Gestión de Solicitudes de Viaticos, Comprobaciones de Gastos Efectuados en Materia de Viaticos, Reportes de Gastos Finalizados y No Finalizados,
•Módulo Almacenes : Consulta de Stock Equipos y/o Materiales Existentes en los diversos almacenes, Reservación de Equipos y/o Materiales Existentes en Stock,
•Módulo Ordenes de Compra : Consulta de Ordenes de Compra Efectuadas por los colaboradores, Consulta de Estatus de Materiales y/o Equipos Solicitados.

Las tecnologias utilizadas para el desarrollo de este sistema web son:
• HTML 5 • CSS 3 • Bootstrap • JavaScript • PHP Versión 8 • Ajax • Laravel Versión 7 • MySQL • Firebird (Sistemas ASPEL)

  • Cliente: AEG Systems S.A. de C.V.
  • Categoria: Diseño y Desarrollo de Software
