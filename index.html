Sistema de Control Administrativo

Diseño, Desarrollo e Implementación de Modulo Plan de Trabajo para la empresa AEG Systems S.A. de C.V.

Este proyecto, consistió en el diseño, desarrollo e Implementación de un módulo adicional dentro del Sistema de Control Administrativo de la empresa AEG Systems S.A. de C.V. El objetivo de este módulo es conocer en plan de trabajo de cada colaborador adscrito a la organización.

Algunas de las funciones implementadas dentro de este módulo fueron:
• Gestión de Tipos: es decir, generar altas, actualizaciones, eliminaciones y consultas de los tipos de plan de trabajo realizada por cada colaborador,
• Gestión de Fases : es decir, generar altas, actualizacion, eliminaciones y consultas de las fases correspondientes a cada tipo de plan de trabajo,
• Gestión de Actividades : es decir, generar altas, actualizaciones, eliminaciones y consultas de las actividades corrrespondientes a cada fase de trabajo ,
•Gestión de Planes de Trabajo : es decir, genera altas, actualizaciones, eliminaciones, consultas, calendarizacion mensual de las actividades en las que se encuentra trabajando cada colaborador adscrito,
•Plan de Trabajo General : es decir consultar el calendario general de la empresa, donde se plasma las actividades en las que esta trabajando cada colaborador adscrito a la organización, asi como exportar los registros del plan de trabajo general a un formato excel.

Las tecnologías utilizadas para el desarrollo del módulo son:
• HTML 5 • CSS 3 • Bootstrap • JavaScript • PHP Versión 8 • Ajax • Laravel Versión 8 • MySQL

  • Cliente: AEG Systems S.A. de C.V.
  • Categoría: DIseño, Desarrollo e Implementación de Módulo
