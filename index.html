ASEAGUA

Diseño y Desarrollo de Sitio Web de Asesoría Jurídica en Materia de Aguas Nacionales - ASEAGUA

Este proyecto, consistio en la implementación de un sitio web estatico para la empresa Asesoría Jurídica en Materia de Aguas Nacionales - ASEAGUA dirigida por el Lic. Juan Alberto Garces. El objetivo de este sitio web fue dar a conocer los servicios ofrecidos por el cliente.

Las actividades realizadas en el proceso de este proyecto son:
• Diseño de Logo Corporativo
• Gestion de Proceso Administrativo (Misión, Visión, Servicios)
• Diseño y Desarrollo de Sitio Web

Las tecnologias utilizadas para el desarrollo de este sitio web son:
• HTML 5 • CSS 3 • bootstrap • JavaScript

  • Cliente: Asesorias Juridicas en Materia de Aguas Nacionales - ASEAGUA
  • Categoria: Diseño y Desarrollo de Sitio Web, Identidad Organizacional, Ilustración

Sistema de Administracion de Personal

Diseño y Desarrollo de sistema en entorno web para Gestión del Agua y Medio Ambiente S.C.

Este proyecto, consistio en la implementación de un sistema web para la empresa Gestión del Agua y Medio Ambiente S.C. dirigida por el Dr. Leonardo Hernandez Barrios. El objetivo de este Sistema es automatizar al 100% la administracion del personal contratado.

Algunas de las funciones que se pueden realizar dentro de esta plataforma son:
• Gestion de Ordenes de Servicio (Registro, Actualizacion, Eliminación, Listado, Carga Masiva, Formato de Registro, Errores de Registro)
• Gestion de Categorias de Trabajo (Registro, Actualizacion, Eliminación, Listado, Carga Masiva)
• Gestion de Reportes de Actividades (Registro, Actualización, Eliminacion y Consulta)
• Gestion de Personal Subcontratado (Registro, Actualización, Consulta, Eliminación, Carga Masiva, Generacion de documentacion de contratacion, Generacion de documentacion de recontratacion)

Las tecnologias utilizadas para el desarrollo de este sitio web son:
• HTML 5 • CSS 3 • bootstrap • JavaScript • PHP • Ajax • Laravel 7

  • Cliente: Gestión del Agua y Medio Ambiente S.C.
  • Categoria: Desarrollo de Software

  • Categoria: Desarrollo de Software

Sistema de Control Administrativo

Diseño y Desarrollo de Sistema Web para la empresa AEG Systems S.A. de C.V.

Este proyecto, consistio en la implementación de un sistema web para la empresa AEG Systems S.A. de C.V. El objetivo de este Sistema es automatizar al 100% algunos procesos del área de Recursos Humanos

Algunos de los modulos existentes dentro de esta plataforma web son:
• Configuración Organizacion (Gestion de Departamentos, Gestion de Puestos de Trabajo),
• Configuración Operacional (Gestion de Permisos, Roles de Usuarios y Usuarios),
• Configuración Laboral (Gestion de Personal Adscrito, Dias de Descansos, Vacaciones, Paternidad, Vencimientos, Pendientes),
• Configuración de Plantas (Gestión de Plantas, Relación Personal Plantas, Requisitos Plantas, Relacion Requisitos Plantas),
• Costo de Nómina (Gestión de ISR, Factor de Intregación, Nómina mensual de colaboradores).
• Ausencias de Personal (Gestion de Solicitudes de Dias de Descanso, Vacaciones, Paternidad),
• Pendientes (Gestión de Documentación / Requisitos de Personal Adscrito)
• Estudios Medicos (Gestión de Estudios Medicos de Personal Adscrito)
• Accesos a Plantas (Gestión de Requisitos y Constancias DC3 por cliente),
• Equipos de Protección de Personal (Gestión de Equipo de Protección Personal a Personal Adscrito por Acceso a Plantas),
• Reservas (Gestión de Citas a sub-áreas de RR.HH.),
• Registro de Horas (Gestión de Registro de Horas laboraras por proyecto asignado.)
• Conocimientos (Gestión de Evaluaciones de Conocimiento a Personal Adscrito, para toma de decisiones),
• Asistencia (Gestión de Registro de Asistencia de Personal Adscrito y Vigente de la empresa),
• Proyectos AEG (Gestión de Proyectos Vigentes y Relación con Colaboradores Adscritos),
• Buzon AEG

Las tecnologias utilizadas para el desarrollo de este sistema web son:
• HTML 5 • CSS 3 • Bootstrap • JavaScript • PHP Versión 8 • Ajax • Laravel Versión 8

  • Cliente: AEG Systems S.A. de C.V.
  • Categoria: Diseño y Desarrollo de Software
